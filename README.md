## How to Run

   ```sh

   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Set up EmailJS:**
   - Copy `.env.example` to `.env` and add your EmailJS keys (see below)
4. **Start the app:**
